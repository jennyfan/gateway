Metagov Gateway

Infrastructure for online communities

The Metagov Gateway is an API gateway for online communities. It connects platforms such as Discord, Slack, and Discourse to each other and to a selection of smaller services, mods, and scripts. Like traditional enterprise gateways, the Metagov Gateway helps you manage a number of different API services.

Built for web3

Metagov Gateway is the foundation on which researchers are building the next generation of online communities.

Made for Governance

Gateway is designed to support rapid prototyping of governance systems, decision-making processes, and social workflows across a range of platforms, from forums to chat services to blockchains. To help people prototype, Gateway ships with a powerful driver for authoring governance policies over multiple platforms.
